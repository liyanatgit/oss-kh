## Tecent Cloud
### Linuxインスタンスの初期設定
 OS選定、インスタンス起動。  
 必要に応じてrootパスワードリセット。
 rootにて、sshログイン。
 管理ユーザ作成、sshのパスワードログイン禁止、rootログイン禁止へ変更し再起動

### CentOS設定手順
+ 1.CVM System Config
 ```
  CentOS 7.2 64bit
  Password: xxx
 ```
+ 2 SSH config to prevent remote password login
+ 3 import SSH public-key and bind to cvm
